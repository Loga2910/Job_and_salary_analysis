#  Jobs and Salary Breakdown Dashboard | Power BI

##  Project Overview

The **Jobs and Salary Breakdown Dashboard** is an interactive Power BI project designed to analyze salary trends across different job roles and locations. The dashboard helps users understand salary distributions, identify high-paying positions, compare compensation across regions and highest job vacancies across regions.

This project demonstrates data visualization, business intelligence reporting, and analytical storytelling using Power BI.

##  Project Objectives

* Analyze salary distribution across job titles.
* Compare average salaries between locations.
* Identify the highest and lowest salary roles.
* Track total job opportunities available.
* Discover salary trends across different countries.
* Enable interactive filtering for detailed analysis.

##  Dataset Features

| Field            | Description               |
| ---------------- | ------------------------- |
| Job Title        | Name of the role          |
| Salary           | Annual salary offered     |
| Location         | Job location              |
| Experience Years | Required experience       |
| Education Level  | Qualification required    |
| Industry         | Industry category         |
| Company Size     | Organization size         |
| Skill Count      | Number of required skills |
| Certifications   | Relevant certifications   |
| Remote Work      | Remote/Hybrid/On-site     |
